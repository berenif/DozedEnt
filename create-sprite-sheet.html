Instructions:

This tool generates a 192x192 pixel sprite sheet for the DozedEnt player character with 6 animation states:

  • Row 0: Idle frames (Green) - 4 frames
  • Row 1: Run frames (Blue) - 6 frames
  • Row 2: Attack frames (Orange) - 4 frames
  • Row 3: Block frames (Dark Blue) - 1 frame
  • Row 4: Roll frames (Yellow) - 4 frames
  • Row 5: Hurt frames (Red) - 2 frames

Usage:

  1. Click "Generate Sprites" to create the sprite sheet
  2. Click "Download Sprite Sheet" to save the file
  3. Save the file as player-sprites.png in the src/images/ directory
  4. Reload your game to see the player animations
